Название: Umschool Extension
Источники: Chome Webstore Firefox Addons
Исходный код: GitHub
ЯП: JS

Простое расширение для сайта https://umschool.net/, которое позволяет смотреть вебинары через YouTube вместо кривого встроенного плеера, а также изменяет некоторые элементы интерфейса для более приятного и удобного пользования сайтом.
Изначально это было скриптом для Tampermonkey, однако для общей удобности был переделан в расширение для браузеров.
Работает как в браузерах на базе Хромиума (Google Chrome, Yandex browser, Opera и т.д.), так и в "Огнелисе" (Firefox).
